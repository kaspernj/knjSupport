Help the TightVNC Project
 
Improve TightVNC:
Make a Donation!

There is a number of ways you can help the TightVNC project.

Make a Donation

The most straightforward way to support our project is to make a donation. In particular, we ask for donations from companies that use TightVNC for commercial purposes. See how to make a donation.

Help Other TightVNC Users

The TightVNC mailing list is a place where people ask for help with TightVNC installation, configuration and usage. If you know the answer to a question appeared in the mailing list, please don't hesistate to send a message to the list. We always appreciate your help in making our mailing list a place where it would be easy to get help.

See the instructions on using TightVNC mailing lists.

Assist in the Development

  • Subscribe to the TightVNC Development mailing list (see instructions here);
  • report bugs and help in their resolution;
  • write more documentation;
  • sponsor development of new features, or compensate developers' efforts to speedup the development.

Let Other People Know

Making TightVNC more popular means more support from the user community, more bug reports and more sponsored major improvements. This all can boost TightVNC development and bugfixing activities, improve the quality of support provided by the community in mailing lists. There are different ways to make the software more popular:

  • placing links to the TightVNC site at your Web pages;
  • rating TightVNC in software directories such as freshmeat.net;
  • recommending TightVNC in Web forums;
  • writing TightVNC reviews for Web sites and printed magazines etc.
